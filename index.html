MCP サーバーヘルプ

MCPサーバーを起動するとAIクライアントから作戦ボードを直接操作できます。

起動:

cd mcp-server
npm install
npm start

ツールバーの ● が緑になれば接続済み。

Claude Desktop 設定 claude_desktop_config.json:

{
  "mcpServers": {
    "football-board": {
      "type": "http",
      "url": "http://localhost:3000/mcp"
    }
  }
}

利用可能なツール:

  • get_board_state — 配置・設定を取得
  • move_player — プレイヤーを移動
  • move_ball — ボールを移動
  • set_formation — フォーメーション変更
  • save_frame — フレーム保存
  • reset_ball — ボールをセンターへ
  • clear_drawings — 描画線を消去
